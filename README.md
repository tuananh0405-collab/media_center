# Media Center

## Project Overview
The Media Center project is designed to provide users with a consolidated platform for managing and accessing various forms of media, including music, videos, and images. It aims to streamline the media experience across devices.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/tuananh0405-collab/media_center.git
   ```
2. Navigate to the project directory:
   ```
   cd media_center
   ```
3. Install the required dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```

## Architecture Documentation
The Media Center utilizes a modular architecture, featuring:
- A front-end built with React for a responsive user interface.
- A back-end powered by Node.js and Express to handle API requests.
- A MongoDB database for storing media metadata.
- An authentication system to ensure secure access.
